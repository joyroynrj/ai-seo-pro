== Description ==

AI SEO Pro is a professional WordPress SEO plugin that leverages artificial intelligence to generate optimized meta tags, analyze content, and improve your website's search engine visibility.

**Key Features:**

* AI-Powered Meta Generation (Anthropic Claude, Google Gemini)
* Real-time Content Analysis
* SEO Score Calculator
* Focus Keyword Optimization
* Readability Analysis
* Schema Markup Generator
* Open Graph & Twitter Cards
* Search Engine Preview
* Custom Post Type Support

**Supported AI Providers:**

* Anthropic (Claude)
* Google (Gemini)

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/ai-seo-pro/`
2. Activate the plugin through the 'Plugins' screen in WordPress

== Frequently Asked Questions ==

= Do I need an API key? =

Yes, you need an API key from Anthropic, or Google to use the AI features.

= Which AI provider should I choose? =

All three providers work well. Claude excels at natural language, and Gemini is Google's newest offering.

= Does it work with custom post types? =

Yes! You can enable AI SEO Pro for any public post type.

= Will it conflict with Yoast SEO? =

We recommend deactivating other SEO plugins to avoid conflicts.
